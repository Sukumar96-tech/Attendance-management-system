# 🎓 College Attendance Management System

## Overview

A full-stack College ERP Attendance Management System built with
**Next.js**, **TypeScript**, and **PostgreSQL**. It enables
administrators to manage students, mark attendance, generate reports,
and allows students to view their attendance history and profile.

## Features

-   🔐 Secure Admin & Student Login
-   👨‍🎓 Student Management (Add, Edit, Delete)
-   ✅ Daily Attendance Marking
-   🟢 Mark All Present / 🔴 Mark All Absent
-   📜 Attendance History
-   📊 Student, Daily, Monthly, and Year-wise Reports
-   📥 CSV Export
-   🖨️ Print Reports
-   ⚙️ Password Change & Settings
-   📱 Responsive UI

## Tech Stack

-   Frontend: Next.js, React, TypeScript
-   Backend: Next.js API Routes
-   Database: PostgreSQL
-   Styling: CSS

## Project Structure

-   app/
-   components/
-   lib/
-   public/

## Installation

``` bash
npm install
npm run dev
```

## Environment

Create a `.env.local` file and configure your PostgreSQL connection.
